Live Crypto Market Dashboard

A Python project that pulls real-time cryptocurrency data via the CoinGecko API 
and analyzes price trends, volatility, and correlation across the top 10 coins.

What it does
- Pulls live market data (price, market cap, 24h/7d change)
- Visualizes 7-day price trends
- Builds a correlation matrix between coins
- Flags anomalies (>5% price moves in 24h)

Tools
Python, Pandas, Plotly, REST API (CoinGecko)
